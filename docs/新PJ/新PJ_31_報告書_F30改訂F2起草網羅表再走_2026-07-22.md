## 1. 起草（1）F-30改訂文案

**この文案はClaude Codeが材料A §3・材料B・材料C・材料Dから合成した草案であり、これまでの
F-27・F-28〜F-31のような「机上（Opus）が既に確定した文言をそのまま転記する」ものではない。
起草の根拠を全て明示するので、設計チャット13側で意味内容の確認・修正を行ってから確定してほしい。**

### 現行文（改訂前・凍結文案）

```
自分の法則への最初の反証は、「法則が人ごとに違う」へ直行させず、まず仲間のミスまたは偶然に
見えなければならない。確信が育つ前に反転させない。
```

### 改訂草案

```
自分の法則への最初の反証は、「法則が人ごとに違う」へ直行させず、二つの逃げ道を経由しなければ
ならない。逃げ道①：自分の持ち場の内側だけで生じた偶然。逃げ道②：他人の持ち場に触れても結果
が返らないことによる確認不能（持ち場境界がその供給元）。両方の逃げ道が尽きるまで、確信は育た
ない。確信が育つ前に反転させない。
```

### 起草の根拠（要確認点を含む）

- **「逃げ道2本化」の解釈：**材料A §3が「F-30の逃げ道『図形が違うからでは』を閉じる」とだけ述べ、
  もう一本の逃げ道の中身を明記していなかったため、Claude Codeが材料B（F-34の「結果は返さない」）
  から「他人の持ち場に触れても結果が返らないことによる確認不能」を第二の逃げ道として合成した。
  **これは推測であり、設計チャット13の意図と一致するか確認が必要**
- 「持ち場境界がその供給元」は【やること】(1)の「持ち場境界の供給元」の文言をそのまま反映した
- 旧文の「まず仲間のミスまたは偶然に見えなければならない」を、逃げ道①（持ち場内の偶然）へ
  部分的に吸収させた。「仲間のミス」という帰属先が新文案から消えている点は**要確認**（削除して
  よいか、逃げ道のどちらかに残すべきかの判断は行っていない）
