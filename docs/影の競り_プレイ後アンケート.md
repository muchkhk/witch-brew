## C. インタビュー用確認

回答後、進行役は次だけを確認する。新しい評価語を誘導しない。

1. 記載された場面の戦番号・原石番号を特定する。
2. 「誰の、何点付近の、どの行動か」を確認する。
3. 結果を見る前に何を予想していたか確認する。
4. 履歴を見せる前の説明を記録する。
5. 履歴確認後に説明が変わった場合、その差を記録する。

「読みが楽しかったですか」「読みが勝利に役立ちましたか」と結論を直接尋ねず、具体場面から判定する。
