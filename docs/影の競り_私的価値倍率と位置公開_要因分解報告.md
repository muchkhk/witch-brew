## 1. 結論

**B2（観測位置公開＋私的価値2倍）を人間テスト候補として支持しない。**

B2混合卓のS2は、S1に対して勝率26.50%対26.61%、平均得点38.40対38.98、1落札利益1.67対3.40で優位を示さず、過払い率は29.85%対14.88%だった。判定基準1～3を満たさない。

位置公開そのものはS2に小幅な正効果を示した。勝率効果は3倍条件で+0.535pt、2倍条件で+0.804ptだった。しかし差の差である交互作用は+0.269pt、95% CI −0.310～+0.849ptでゼロをまたいだ。平均得点、落札利益、過払い率の交互作用もほぼゼロだった。**2倍条件で位置公開が3倍条件より特に有効になる証拠はない。**

2倍化だけの効果は、非公開条件のS2勝率で+4.69pt、公開条件で+4.96ptと大きかった。一方、評価される私的価値自体が小さくなるため平均得点と1落札利益は低下し、S2の過払い率は約2.5pt増えた。B2の変化を位置公開との相乗効果とは解釈できない。

- 独立10シード群、各ミクロセル250ゲーム。

## 6. 問いごとの回答

1. **2倍化だけで改善するか：** S2勝率はA3→A2で+4.69pt。ただし平均得点・落札利益は尺度縮小もあり低下し、過払い率は+2.51pt。全面的改善ではない。
2. **3倍条件で位置公開が悪化要因か：** S2単体には小幅改善。勝率+0.535pt、利益+0.187、過払い−1.665pt。
3. **2倍条件で位置公開が有益か：** S2単体には小幅改善。勝率+0.804pt、利益+0.166、過払い−1.648pt。ただしS1優位を覆さない。
4. **B2改善は位置公開か2倍化か：** 勝率変化の大半は2倍化。位置×倍率の交互作用は小さくCIがゼロをまたぐ。
5. **私的価値が弱すぎないか：** 死滅とは断定できないが、2倍条件ではS2の利益が1.50～1.67、過払いが約30%となり、3倍条件より余剰が薄い。人間テスト候補に進める前の警報である。
6. **S2はS1より利益を得るか：** B2で得ない。利益1.67対3.40、得点38.40対38.98、勝率26.50%対26.61%。
7. **読み強化で過払いが増えるか：** 同一倍率で位置公開するとS2過払いは約1.65pt減った。一方、2倍化すると約2.5pt増えた。

## 7. B2の判定

| 判定基準 | 結果 |
|---|---|
| S2勝率・得点がS1より同方向に改善 | **不合格**。勝率−0.11pt、得点−0.58 |
| 落札利益が改善 | **不合格**。S2 1.67、S1 3.40 |
| 過払いが大幅悪化しない | **不合格**。S2 29.85%、S1 14.88% |
| Reader-OnがOffより一貫して良い | **一部合格**。得点・利益・過払い・MAEは改善、勝率CIはゼロをまたぐ |
| 2倍時の位置効果が3倍時より良い | **弱い方向のみ**。勝率交互作用+0.269ptだがCIはゼロをまたぐ。他指標はほぼ同じ |
| 改善が2倍化だけではない | **不合格**。大きな勝率変化は2倍化、交互作用は不明確 |

総合判定：**支持しない。B2を人間テスト候補にしない。**

## 8. 統計処理

- 10個の独立シード群をクラスタとした。
- 各シード群内でA3/B3/A2/B2およびReader-On/Offへ共通乱数を使用。
- 差は各シード群の平均指標を先に引き算し、10個の対応差から平均・シード間SD・95% CIを算出。
- 95% CIは自由度9のt値2.262を使用。
- JSONには各シード群の生値、シード間SD、対応差CIを保存。
- 各条件の勝率には参考として全ゲーム二項近似CIも保存したが、主要判断はシード群対応差CIを使用。
- 同率首位は同率人数で1勝を分割した。

## 9. 検証上の制約

- S2は事前指定された単一構造、α=0.25固定。別の合理的読みモデルは未検証。
- S3は現行NPCの読み式を共通基準として移植したが、製品の席別プロフィール差は付けていない。
- シミュレーション戦略は人間のブラフ、癖、誤読、学習を表現しない。
- 2倍化は得点価値の尺度も変えるため、倍率をまたぐ平均得点・利益の絶対差は単純な効用改善ではない。
- Reader-Offも過去履歴から位置確率を推定する。完全な情報無視ではなく、公開位置だけのアブレーションである。
- 10シード群のCIは有限標本。小差を製品判断へ一般化しない。
- 面白さ、緊張感、私的価値の体感、疑心暗鬼は判定不能。
