## 1. 結論

**観測位置公開単独の案B（後続検証のB3）は支持されなかった。** Bは推定精度を改善したが、局所利益・勝率への十分な優位へ変換されなかった。一方、この検証の現行A混合卓ではS2が勝率29.36%・平均得点41.94で全戦略中最高だった。ただし後続A3検証では同様の優位を再現していない。**現行Aでは履歴読みが有効になるモデルも存在するが、その優位は戦略実装と対戦構成に敏感で、頑健性は未確認**と整理する。

当初の比較対象Cは後続2×2検証のB2として実施した。B2はS1に対して勝率・得点・落札利益で優位を示さず、過払いも高いため支持しない。A2・B3も採用せず、次の人間テストまでは現行A3を維持する。

Bは標準4人混合卓でS2の総価値推定MAEを2.895から2.793へ改善した。しかしS2勝率26.99%はS1の27.62%を上回らず、95%信頼区間も重なる。推定精度の改善は最終勝利へ変換されなかった。B-liteは平均1.86個目で全員の位置が判明し、ほぼ早期B化した。B-historyは十分な位置別標本が全位置に2件ずつ溜まったゲームが44.4%だけで、情報が遅く弱かった。

判定は次の通り。

| 案 | 判定 | 要点 |
|---|---|---|
| A | 基準・維持候補 | S2は混合卓で勝つが、推定誤差は改善していない。モデル化した相手の搾取を含む |
| B | **一部支持／人間比較テスト候補** | 推定精度は改善。局所利益・勝率への変換は不支持 |
| B-lite | **不支持寄り** | 位置公開コストがほぼ残らず、2個目前後でBになる |
| B-history | **不支持** | 匿名情報が遅く、S2はS1を上回らない。人物推理は残るが改善効果が弱い |
| C（位置公開＋私的価値2倍、後続B2） | **後続検証で不支持** | 小さな位置効果はあったがS1を上回る利益や倍率との交互作用なし |

シミュレーションだけで「面白い」「製品版へ採用」とは判定しない。

## 2. 実ルール表

実装上の正として `seri.html` v1.4を抽出した。

| 項目 | 実装 |
|---|---|
| 2人 | 3面、原石6、初期資金30 |
| 3人 | 3面、原石6、初期資金30 |
| 4人 | 3面、原石8、初期資金34 |
| 5人 | 4面、原石10、初期資金38 |
| 6人 | 4面、原石12、初期資金38 |
| 面 | 色味、透明度、輝き。5人以上は大きさを追加 |
| 面値 | 各面独立に整数−2～+6の一様分布。事前期待値2 |
| 観測位置 | ゲーム開始時に各人が独立抽選。重複あり、ゲーム中固定 |
| 原石共通価値 | 全面の合計 |
| 個人価値 | 全面合計 + `2×自分の観測面`。したがって観測面は合計3倍相当 |
| 得点 | 残金 + 落札した原石の個人価値合計 |
| 勝敗 | 得点降順。同点を解消する追加規則はコードにない。本検証は同率首位を人数割り |
| 開始者 | 各原石ごとに資金1以上の者から一様ランダム |
| 刻み | 1。自分の番は現在価格+1で入札するか降りる |
| 降り | 当該原石では不可逆 |
| パス | 独立したパスはなく「降りる」。誰も価格を上げなければ流札 |
| 槌 | 生存者1人、または手番が最後の入札者へ一巡した時点 |
| 落札 | 落札者の資金から価格を引き、原石を取得。全面と共通価値を公開 |
| 資金不足 | 次の1刻みを払えなければ強制的に降りる。次原石では資金1未満の者を除外 |
| 現行NPC基礎 | 自面+未知面期待値を共通価値推定とし、私的加算、読み、呪い控除、楽観係数を適用 |
| NPCプロフィール | 席別に `(opt,read,curse)` = (1,1,1), (1,1,1), (.85,1,1.4), (1.1,.3,.3), (.95,1,.8), (1.05,.6,1.1) |
| ウィナーズ・カース控除 | `curse × 生存競争相手比率 × 2 × (面数−1) × 0.42` |
| 履歴から位置推定 | 各位置仮説に `熱意×(公開後の面値−2)` を累積。熱意=`その人の最高入札−共通価値/2`。softmax温度12 |

## 3. 検証モデル

- 製品の生成、資金、得点、上り競り、開始者、強制降り、槌、現行NPC式を移植した。
- A/B/B-lite/B-historyは同じ原石列・観測位置列を使う固定シード比較。
- 標準4人卓：4ルール×9対戦構成×10,000ゲーム。
- S2感度：4ルール×係数0.25/0.50/0.75/1.00×10,000ゲーム。
- 補助：2・3・5・6人×4ルール×混合卓5,000ゲーム。
- 合計68セル、延べ600,000ゲーム超。
- 混合卓はS0/S1/S2/S3を席順ローテーション相当で配置。対現行NPC、全員同戦略も別に実施。
- S0は自面、未知面期待値、資金、呪い控除のみ。S1は人数・平均・最高入札だけを使う。S2は同位置信号を重複計上せず、異なる公開位置を別証拠として評価する。S3は現行NPC。S4は終盤・首位候補生存・継続可能性・誤落札耐性・他生存者ありの条件だけで1～2段上げる。
- 「ほぼ確定」は機械的代理として、3面卓なら異なる高値位置が2つ以上、4面卓なら3つ以上ある原石と定義した。心理的確信とは異なる。

## 6. 戦略別結果

- 対現行NPCの4人卓ではS2勝率はA 27.71%、B 27.96%、B-lite 27.71%、B-history 27.66%。差は誤差範囲。
- 対称卓は全戦略で席勝率25%となり、進行不能や席支配はなかった。
- 対称S2の平均得点はA 40.82、B 40.22、B-lite/B-history 40.12。公開で総利益は改善せず、競りの過熱が示唆される。
- 限定S4は約0.25回/ゲームだけ発動。対現行NPC勝率はA 27.32%、B 27.54%、B-lite 27.33%、B-history 27.40%。対応S2との差は小さく、勝率改善とは判定できない。
- S4の誤落札率は約15～17%。今回は安全条件が厳しく、誤落札時損益も正になった。これは「ブラフが一般に強い」のではなく、ほぼ価値入札に近い限定局面だけが残った結果である。

## 8. 「読みが報われた」3層分析

### 層1：推定精度

Bは支持、B-liteは一部支持、B-historyは不支持。標準4人混合卓のS2はS0比でB +0.119、B-lite +0.088、B-history −0.006点のMAE改善。

### 層2：局所利益

不支持。S2の落札純利益はA 4.92からB 3.91、B-lite 3.76へ悪化。BではS2の呪い率が20.5%で、S1の15.1%より高い。精度向上で追える石が増えた一方、公開情報が価格にも反映され、余剰が消えた。

### 層3：最終勝利

不支持。B/B-lite/B-historyの混合卓でS2はS1を上回らず、対現行NPCでもAとの差は信頼区間内。推定精度→局所利益→勝利の変換鎖は層1で止まった。

## 9. 破綻チェック

### B

- 答えが簡単すぎる：代理基準の「ほぼ確定」信号は4人卓で34.4%の原石。低くはなく、人間の認知上は要確認。
- 高値位置を数えるだけ：S1がS2以上なので危険は現実化。細かな位置推論の追加価値が勝利へ出ていない。
- 人物推理：開始時公開なので位置同定の長期推理は消える。
- S2支配：起きていない。むしろ5～6人でS1に負ける。

### B-lite

- 最初の入札で公開されたプレイヤー率99.995%、最後まで非公開0.005%。
- 全員判明ゲーム99.98%、判明は平均1.86個目。
- 情報秘匿のための不自然な降りを明示的に最適化する戦略は未実装。それでも通常戦略では公開コストがほぼ消滅した。
- 現仕様では独立案というより「最初の約1～2石だけA、その後B」。

### B-history

- 全位置に2標本以上が溜まるのは44.4%のゲームのみ。到達時は平均3.24個目。
- 個人名との対応を隠すため人物推理は残るが、現在の相手の入札へ位置を結びつけられず、S2優位が出ない。
- 表示情報量に対して実利が小さい。匿名集計からの逆算と認知負荷は人間テスト前に懸念。

### 共通

- 2人では異なる複数位置信号が少なく、公開の意味は薄い。
- 5～6人では情報量が増え、S2がS1へ3～5pt負ける。
- ウィナーズ・カースは残るが、公開S2で消えるのではなく、読みの価格反映により再増加した。
- 終盤の資金計算支配、B-liteの意図的秘匿、匿名履歴の人間による逆算は本モデルだけでは判定不能。

## 10. 人間テスト候補

最初に行うのは、標準4人・現行A3・同じ参加者による3～5戦の人間テストとする。B2・B3・A2は採用せず、降りを負のシグナルとして認識できるか、同じ相手から学習するか、読みで判断が変わり面白さや納得へつながるかを確認する。

1. Bで「誰がどの面か」を考えず、異なる高入札位置の数だけを数えなかったか。
2. 推定が良くなった結果、支払額も上がって利益が消えなかったか。
3. Aで存在した人物ごとの位置推理がなくなり、物足りなくなったか。
4. Bで答えを確信できる石は何個あったか。その確信は実際に当たったか。
5. 同じ面の複数入札を重複証拠として扱わない判断が自然にできたか。
6. 自分情報だけで勝つ道が残ったか。
7. 終盤は読みより資金・順位計算が支配したか。
8. 低い面から首位を1～2刻み押す限定ブラフが実際に合理的だったか。
9. A/Bのどちらが「読めるが、読み切れない」と感じたか。
10. 再戦時にも同じ判断が残るか、閾値戦略へ収束したか。

## 11. 限界

- 当初の比較対象Cは本検証時点では未検証だったが、後続2×2要因検証のB2として実施済み。
- 混合卓はS0/S1/S2/S3を固定席へ配置し、原石ごとの開始者だけをランダム化した。完全な席ローテーションではない。
- 固定シード系列は用いたが、独立した複数シード群の間の差は測っていない。
- A/Bは共通乱数だが、対応差そのものの信頼区間は算出していない。
- S2は係数感度を変えたが、読みモデルの構造は一種類だけである。別の合理的読みモデルで結論が変わる可能性がある。
- S0～S4は完全均衡ではなく、コード化した合理的ヒューリスティック。特にS2の信号逆算誤差は現行NPCの読み・楽観度差を完全には同定しない。
- B-historyの匿名履歴利用は保守的。より複雑な統計モデルなら改善し得るが、それは人間が実戦で使える複雑さを超える可能性がある。
- B-liteで「公開したくないから一度も入札しない」を目的関数に含む専用戦略は実装していない。通常戦略下の公開速度を測った結果である。
- 95%区間はゲームを独立標本とする正規近似。共通乱数による対応差の専用区間は算出していない。
- 「ほぼ確定」の代理定義は情報量の上限警報であり、人間の確信を直接測らない。
- 面白さ、緊張、疑心暗鬼、人物読みの満足、ブラフの対人効果は判定不能。

## 12. 次の推奨と最終質問への回答

1. **B3・B2：不支持。A2も不採用。** このA混合卓ではS2が最高だったが、後続A3で再現せず頑健性未確認。
2. **S2はS1より本当に優れるか：全体として不支持。** B系4～6人、とくに5～6人で劣る。
3. **精度改善は利益・勝率へ変換されたか：されなかった。**
4. **Bは情報過多か：危険あり。** 約34%の石に強い複数位置信号があり、S1で十分という結果。
5. **B-liteの公開コスト：成立していない可能性が高い。** 平均1.86石で全公開。
6. **B-history：人物推理は残すが遅く弱い。**
7. **2～6人で同じ案：推奨しない。**
8. **最初の人間テスト：4人・現行A3を3～5戦。** 負の読みを人間が認識・実行し、判断と体験へつながるか確認する。
9. **人間だけで確認できる点：**楽しさ、読み切れなさ、人物推理喪失、認知負荷、限定ブラフ、再戦性。
10. **A維持の可能性：十分ある。** 現段階では最も保守的で合理的な結論。
