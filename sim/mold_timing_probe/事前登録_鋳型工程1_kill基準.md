## 0. 目的

開示が無料で自動進行する世界で、「いつ賭けるか」に市場が立つかを、ゲーム未実装の裸モデルで測る。判定はすべて本文書の規則による機械判定。事後的な閾値再解釈・敗者復活は行わない。蘇生試行の事前用意なし——killなら死亡記録へ直行する。

## 1. 語彙表（判定の分母となる単位語・凍結）

| 語 | 定義 |
|---|---|
| 1ディール＝1試行 | 両ダイスの鋳造 → 開示ステップ1..6 → 精算まで |
| 開示ステップt | t=0は開示前。ステップtで両ダイスの面tが**全員に同時公開**される |
| 賭け | 1ディールに**必ず1回**、時点t∈{0,1,2,3,4,5}で実行する（時点tとは「開示1..tを観測済み・開示t+1の前」。賭けない選択は存在しない＝タイミングのみが選択） |
| 勝ち | 自分の6面合計＞相手の6面合計。同値は勝ち度0.5 |

## 2. モデル（凍結）

- ダイス2個（自分／相手）。各ダイスは独立に50%で鋳型H={4,5,6,7,8,9}またはL={1,2,3,4,5,6}。割当は非公開（所有者も知らない）
- 各面の値：その鋳型の6値から独立一様抽選（**復元あり＝重複あり**）
- 開示順：面1→2→…→6の固定順
- 賭けの対象：戦略自身の事後で勝ちが有利な側に固定額1（事後がちょうど0.5なら自分側——凍結タイブレーク）
- 利得：m(t)×W − 1（W∈{0, 0.5, 1}）。対家精算＝相手の賭け行動は自分の利得に影響しない（純粋計測）

## 4. 戦略の名指し（凍結）

| 戦略 | 規則 |
|---|---|
| S-early | 常にt=0で賭ける |
| S-wait | 常にt=5で賭ける |
| S-th65 | 真の生成モデル（鋳型ベイズ）で勝ち事後を計算し、有利側の事後が**0.65**を初めて超えたtで賭ける。t=5まで超えなければt=5 |
| S-mold | どちらかのダイスの鋳型事後（HまたはL）が**0.85**を初めて超えたtで賭ける。超えなければt=5 |
| S-blind | 各面を真の周辺分布（値1..3および7..9は各1/12、値4..6は各1/6）からの**i.i.d.とみなし、鋳型相関を使わない**事後を計算する。賭け規則はS-th65と同一（閾値0.65） |

S-blindはK-A3の対照であり、「隠れ変数（鋳型）推定の寄与」だけを分離する。

## 6. kill条件（機械判定・凍結）

以下のいずれか1つでも発動したら案Aをkillする。

| | 発動条件 |
|---|---|
| **K-A1（待ち屋支配）** | 3曲線すべてで、X∈{S-early, S-th65, S-mold}のいずれについても、対応差CI95下限(X − S-wait) ≤ 0 |
| **K-A1'（情報無価値）** | 3曲線すべてで、X∈{S-wait, S-th65, S-mold}のいずれについても、対応差CI95下限(X − S-early) ≤ 0 |
| **K-A2（曖昧帯が装飾）** | 各ダイス個体について「鋳型事後P(H|そのダイスの開示1..t)が閉区間[0.2, 0.8]に入っているtの個数（t=1..6中）」を数え、その全個体中央値が **2未満** |
| **K-A3（隠れ変数が装飾）** | 3曲線すべてで、対応差CI95下限(S-th65 − S-blind) ≤ 0 |

- どれも発動しなければ**Pass**。参考記録として「条件付き戦略最良と無条件戦略最良の差」が最大だった曲線を記す（判定には使わない）
- Passした場合の含意は「タイミング市場の原料が実在する」まで。工程2（賭け構造・経済）の保証ではない

## 9. 実装時に発見した仕様の欠落（2026-07-16・実装前に発見・ユーザー確認済み）

**§8の中止条件に該当する欠落を実装開始前に発見し、いったん実装を停止して報告した。**

**発見した欠落：** §2「賭けの対象：戦略自身の事後で勝ちが有利な側に固定額1」の「戦略自身の事後」は、S-th65（真の生成モデル）とS-blind（周辺分布モデル）にしか明示的な定義がない。S-early・S-wait・S-mold には、賭けの**方向**（自分側／相手側のどちらに張るか）を決めるための事後モデルが本文書のどこにも明記されていなかった。

**確定した補完規則（ユーザー確認済み）：** S-blind以外の全戦略（S-early・S-wait・S-mold）は、方向決定に真の生成モデル（鋳型ベイズ）の勝ち事後Pを流用する。S-mold固有の「鋳型事後P(H/L)」は賭けの**タイミング**判定にのみ使用し、方向決定には使わない。

この補完規則は事前登録の凍結後に発見された欠落の補完であり、本文書§2の「凍結」原則そのものを事後変更するものではない（欠落箇所の一意な補完であり、既定義部分の再解釈ではない）。
