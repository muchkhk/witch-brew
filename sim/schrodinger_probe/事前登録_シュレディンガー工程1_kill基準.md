## 0. 目的と主判定の所在

締切つき覗きの原料実在検証。ゲーム・経済・価格は実装しない（sim/info_geometry_probe/ の手法を流用する純粋測定）。

覗きの価値には2チャネルある：①**即時**（自分の面kを知ると面kイベントの見立てが良くなる——どんな隠し面ダイスにもある）②**波及**（全面相異ゆえに、面kを知ると残り面の演繹が進む——本案固有）。**主判定は②に置く。** ②が細ければ本案は「ただの自札隠匿」に還元され、掘る理由が消える。①は参考記録。

判定は機械。事後的閾値再解釈・敗者復活なし。蘇生試行の事前用意なし。

## 1. 語彙表（凍結）

| 語 | 定義 |
|---|---|
| 1ディール＝1試行 | 両ダイス生成 → 面1..6の逐次解決まで |
| 解決ステップk | ステップkで両ダイスの面kの値が**全員に同時公開**される |
| 公開履歴（時点k） | 解決済みの面1..k−1の値（両ダイス分） |
| 覗き | 解決ステップkの**直前**に、自分の面kの値を知る操作。本測定では毎回実施・価格なし |
| 面イベントE_j | 「自分の面jの値＞相手の面jの値」。確率は同値を0.5勝として計算：p = P(＞) + 0.5·P(＝) |
| 反転 | ある見立ての更新で、有利側が入れ替わること：I[(p_前−0.5)(p_後−0.5) < 0] |

## 2. 面値セット（凍結）

各ダイスは6面。生成規則：

| | 生成規則 | 狙い |
|---|---|---|
| D1 純順列 | {1,2,3,4,5,6}の一様ランダム順列 | 演繹最強の極（構成既知・配置未知） |
| D2 広構成 | {1..12}から6値を非復元一様抽選し、一様ランダム配置 | 構成も配置も未知 |
| D3 二峰 | {1,2,3,10,11,12}の一様ランダム順列 | 80/20設計の極 |
| F0-D 死亡対照 | 各面を{1..6}から**独立・復元あり**で一様抽選（相異制約なし） | 理論上waviation≈0のノイズ床。**参考記録のみ・判定に使わない** |

両ダイス（自分／相手）は独立に同一規則で生成。

## 3. 指標定義（凍結）

時点k（解決前）に、2通りの条件付けで各イベント確率を計算する：
- p_a(j|k) = P(E_j | 公開履歴)
- p_b(j|k) = P(E_j | 公開履歴 ＋ 自分の面kの値)

**M2'-imm**（即時・参考）＝ ディール内で k=1..6 の平均：反転I × |p_b(k|k) − p_a(k|k)|

**M2'-prop**（波及・主判定）＝ ディール内で k=1..5 の平均：Σ_{j=k+1..6} [ 反転I_j × |p_b(j|k) − p_a(j|k)| ]

全ディール平均を各セットの値とする。CI95%はブートストラップ10,000回。

## 4. 事後計算の方式（凍結）

- 両ダイスは独立なので、E_jの確率は「自分の面jの値の周辺分布」×「相手の面jの値の周辡分布」の畳み込みで計算する（自分側のみ覗き条件を追加）
- D1・D3：整合する配置の**厳密列挙**（≤720状態/ダイス）
- D2：構成×配置の厳密列挙が重い場合、**整合補完のモンテカルロ近似（1測定点あたり2,000サンプル以上）を許可**。採用した方式と近似の扱いを結果MDに記録する

## 5. 試行数（凍結）

## 6. kill条件（機械判定・凍結）

**K-D0（演繹チャネルが装飾）：** 3セット（D1, D2, D3）**すべて**で、

> CI95下限( M2'-prop ) ＜ 0.25 × 点推定( M2'-imm )（同セット内比較）

が成立したら発動 → 案Dをkill。

- 1セットでも「CI95下限(M2'-prop) ≥ 0.25 × 点推定(M2'-imm)」なら**Pass**
- Passセットが複数の場合、工程2へ持ち越すのは **CI95下限(M2'-prop)が最大の1セット**（機械的タイブレーク）
- Passの含意は「相異制約による演繹の原料が実在する」まで。工程2には死因6のK1（利ざや）がそのまま待っている
